VScript Carjack v1.2.0

Documentation Complète

Vol de véhicule sur commande avec appel cinématique & traqueur GPS

Installation

  1. Téléchargez vscript_carjack
  2. Glissez le dossier dans le répertoire resources de votre serveur
  3. Ajoutez ensure vscript_carjack à votre server.cfg
  4. Redémarrez votre serveur

Fonctionnalités

Intro Cinématique

Appel téléphonique immersif avec séquences caméra

Traqueur GPS

Traqueur avec bips révélant la position à la police

Garage de Hack

Mini-jeu pour désactiver le traqueur avant livraison

Multi-Framework

Support ESX, QB, Qbox & Standalone

Configuration

Paramètres Généraux

Config.Inventory = 'auto'   -- auto, ox, ps, core
Config.Dispatch = {
    enabled = true,
    system = 'lb-tablet',  -- lb-tablet, ps-dispatch, cd_dispatch, etc.
    delay = 3,              -- Delay before police alert
    maxHackAttempts = 2     -- Alert after X failed hacks
}

Véhicules & Récompenses

Config.TargetVehicles = {
    { model = `adder`,    plate = "LUXE001", value = 8000, name = "Adder" },
    { model = `zentorno`, plate = "SPEED77", value = 7500, name = "Zentorno" },
}

Config.Pay = {
    moneyType = 'black_money',
    minHealthPercent = 0.3,
    perfectBonus = 500,
}

La récompense dépend de l'état du véhicule. Un état parfait donne un bonus.

Customisation Avancée

1. Bridge System

Le script utilise un bridge pour gérer les items et l'argent sur tous les frameworks. 

-- customization/inventory.lua
function Bridge.AddItem(source, item, count)
    -- Your inventory logic here
end

2. Dispatch Bridge

Personnalisez l'envoi des alertes police dans customization/dispatch.lua.

Utilisation

1

Lancer la Mission

Interagissez avec la cabine téléphonique pour recevoir votre commande. Une cinématique se lance.

2

Trouver & Voler

Trouvez le véhicule dans la zone. Le voler déclenchera le traqueur GPS.

3

Désactiver le GPS

Foncez au garage de hack. Réussissez le mini-jeu pour arrêter d'envoyer votre position.

4

Livraison

Allez au point de livraison et donnez le véhicule au contact pour être payé.

Dépannage

La cinématique semble cassée

• Vérifiez que vous n'avez pas déplacé les coordonnées de la cabine dans client/main.lua • Vérifiez si un autre script entre en conflit avec le contrôle des caméras

Les alertes ne montrent pas le modèle/plaque

• Vérifiez que vous utilisez un système de dispatch compatible (comme lb-tablet ou ps-dispatch) • Vérifiez le mappage dans customization/dispatch.lua
